Detailed Description

Simulates interrupts on an Arduino Mega (Wokwi project).

This project demonstrates how to simulate external and timer interrupts using an Arduino Mega. It uses three buttons (BUTTON0, BUTTON1, BUTTON2) to trigger external interrupts and a timer (via Timer1) for periodic events. The program also measures distance using an ultrasonic sensor and triggers a distance alert if an object is too close.

Various LEDs indicate the interrupt events and sensor/timer actions. Additionally, several digital pins are toggled to simulate signals for a logic analyzer.

Note
This code is designed for use with the Wokwi simulation environment.
